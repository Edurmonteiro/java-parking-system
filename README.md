# 🚗 Sistema de Estacionamento em Java

Projeto desenvolvido em Java para praticar conceitos fundamentais de programação através da simulação de um estacionamento com 9 vagas.

## 📋 Funcionalidades

* Estacionar veículos em vagas específicas
* Remover veículos do estacionamento
* Visualizar todas as vagas ocupadas e livres
* Buscar veículo pela placa
* Controle automático de vagas disponíveis
* Validação de posições inválidas

## 🛠️ Tecnologias Utilizadas

* Java
* Scanner para entrada de dados
* Arrays e Matrizes
* Estruturas Condicionais
* Estruturas de Repetição

## 📚 Conceitos Praticados

Este projeto foi criado com o objetivo de reforçar conhecimentos em:

* Lógica de programação
* Manipulação de matrizes
* Controle de fluxo (if/else e switch)
* Laços de repetição
* Busca em estruturas de dados
* Validação de entradas
* Organização de código

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Acesse a pasta do projeto:

```bash
cd nome-do-projeto
```

3. Compile o arquivo:

```bash
javac Parking.java
```

4. Execute o programa:

```bash
java Parking
```
